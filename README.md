# AI-Powered-Skincare-Assistant
AI-powered web application for acne detection and skincare insights using CNN and environmental data

Overview
Short explanation of the project.

Features
- Acne classification using CNN
- Image upload through Flask interface
- Basic skincare recommendations
- UV index awareness

Tech Stack
Python, TensorFlow, Flask, CNN

Installation
pip install -r requirements.txt

Run the App
python app.py

Dataset
The dataset used for training contains categorized acne images divided into train, validation, and test sets. 
Due to size limitations, the full dataset is not included in this repository.
